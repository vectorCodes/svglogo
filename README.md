svglogo.dev is a tiny browser tool for idiomatic icon-based logos.

Key behaviors:

- live icon canvas with gradient/solid backgrounds, border radius, and export-friendly outline control
- detachable control dock with tooltips, keyboard shortcuts (undo/redo, icon picker, copy/paste), and a dice shuffle
- analytics-backed tracking for downloads via `umami.track`
- lightweight meta + sitemap for Google crawlers

Run `bunx dev` locally and open the dishwasher-friendly UI in your browser.
